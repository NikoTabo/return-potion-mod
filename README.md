# Return potion
Forge мод на зелья возврата на точку респавна для Minecraft.

Мод добавляет два вида зелий:<br/>
  - `Зелье йо-йо` телепортирует игрока на точку респавна на 30 секунд (1:30 для улчшенного редстоуном варианта), после чего телепортирует в точку, в которой игрок выпил зелье;<br/>
  - у `Зелья возврата` два использования. При первом использовании зелье телепортирует на респавн, при втором телепортирует на точку выпивания зелья.<br/><br/>

Крафты производятся на варочной стойке. <br/>
  - Чтобы сделать `зелье йо-йо`, нужно в качестве основы использовать мутное зелье, а в качестве ингридиента жемчуг эндера. При этом зелье можно улучшить, использовав красную пыль в качестве ингридиента, `зелье йо-йо` в качестве основы.<br/>
  - Чтобы сделать `зелье возврата`, нужно в качестве основы использовать `зелье йо-йо`, а в качестве ингридиента плод хоруса.<br/>
