# Return potion
Forge мод на зелья возврата на точку респавна для Minecraft.

Мод добавляет два вида зелий:
  — одно телепортирует игрока на точку респавна на 30 секунд (1:30 для улчшенного редстоуном варианта), после чего телепортирует в точку, в которой игрок выпил зелье;
  — у другого зелья два использования. При первом использовании зелье телепортирует на респавн, при втором телепортирует на точку выпивания зелья.
